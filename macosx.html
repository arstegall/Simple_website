Mac OS X

Apple Inc. američka je tvrtka osnovana 1976. godine danas prepoznatljiva po kvalitetnim, ali i iznadprosječno skupim proizvodima. Apple je nekada znatno doprinjeo revoluciji osobnih računala kroz svoja Apple I i Apple II računala, a nešto kasnije i kroz Machintosh računala. Mac OS X sustav je koji se temelji na UNIX-u, a tvrtka na čijoj je tehnologiji razvijen kupljena je od strane Applea 1996. godine. Prva objavljena verzija je bila Mac OS X Server 1.0 1999., a desktop verzija Mac OS X 10.5 izlazi dvije godine kasnije. Verzije 10.8 i novije nose naziv OS X. OS X deseta je verzija sustava za Mac računala, dok su se prijašnje verzije označavale arapskim brojevima, sada je slučaj da se koristi rimski broj. Jezgra Mac OS-a u skladu je s POSIX standardom za definiranje sučelja za programiranje te je izgrađena na XNU jezgri. Sadrži standardne mogućnosti UNIX-ove komandne linije. Mac OS X bolji je, sigurniji i stabilniji sustav nego njegov prethodnik Mac OS 9. Primjerice, višezadaćni rad i zaštita memorije omogućuje izvođenje više aplikacije odjednom bez njihovog međusobnog prekidanja.
